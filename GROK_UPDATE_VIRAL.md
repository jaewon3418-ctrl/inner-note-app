## 현재 진단: "괜찮은 앱" vs "터지는 앱"

| 구분 | 설명 | DeepLog 현재 |
|------|------|--------------|
| 나쁜 앱 | 크래시, 못생김, 쓸모없음 | ❌ |
| **괜찮은 앱** | 작동함, 깔끔함, 유용함 | ✅ **여기** |
| 좋은 앱 | 매일 쓰고 싶음, 추천함 | △ 부족 |
| 터지는 앱 | "야 이거 해봐" 바이럴 | ❌ 아직 |

### 왜 "괜찮은 앱"에 머물러 있나?

1. **"와!" 순간 없음** - AI 분석이 일반적, 소름 끼치는 순간 부족
2. **공유 불편** - 예쁜 공유 카드 없음, 주간 리포트만 공유 가능
3. **다시 올 이유 약함** - 하루 1회 제한, 할 거 하면 끝
4. **첫 30초 밋밋** - 앱 열면 빈 텍스트 입력창, "뭘 써야 하지" 막막함

## 구현 계획: 7가지 업그레이드

### 1. 공유 카드 (바이럴 핵심) ⭐⭐⭐
```
┌─────────────────────────┐
│     오늘의 내 마음       │
│    😢 슬픔 ████░ 4/5    │
│  "많이 지쳤구나.        │
│   오늘 하루 버틴 거     │
│   대단한 거야."         │
│      ─ DeepLog AI ─     │
│  2025.01.21 • 속마음노트 │
└─────────────────────────┘
→ 인스타 스토리 1탭 공유
→ 친구가 보고 "이거 뭐야?" → 다운로드
```
- **예상 시간**: 4-6시간
- **이미 있는 것**: react-native-view-shot, expo-sharing

### 2. 첫 경험 개선 (이탈 방지) ⭐⭐⭐
```
현재: 앱 열기 → 빈 텍스트 입력창 → 이탈
개선: 앱 열기 → 이모지 6개 선택 (2초) → AI 즉시 반응 (3초) → "더 이야기할래?"
```
- **예상 시간**: 3-4시간
- **이미 있는 것**: EmotionWheel.jsx 컴포넌트

### 3. AI "소름" 순간 (프롬프트 개선) ⭐⭐
```
현재: "힘든 시간을 보내고 있구나. 내가 옆에 있을게."
개선: "오늘 누군가한테 서운했지? 말 안 해도 느껴져."
→ "어떻게 알았지?!" 반응 유도
```
- **예상 시간**: 1-2시간
- **수정 파일**: src/services/openai.js 프롬프트

### 4. 스트릭 시각적 보상 ⭐⭐
```
현재: "7일 연속 기록 중 🔥" (텍스트만)
개선: 불꽃 애니메이션 + "꾸준함의 증거" 뱃지 + 공유 버튼
```
- **예상 시간**: 4-5시간
- **이미 있는 것**: Lottie 애니메이션 라이브러리

### 5. AI가 먼저 말 걸기 ⭐⭐
```
현재: 사용자가 먼저 입력해야 함
개선: 앱 열면 "어제 슬프다고 했잖아. 오늘은 좀 어때?"
```
- **예상 시간**: 3-4시간
- **필요한 것**: 최근 감정 히스토리 분석 로직

### 6. 데일리 챌린지/질문 ⭐
```
현재: 매일 같은 화면
개선: 월요일 "이번 주 기대되는 건?", 화요일 "어제 행복했던 순간?" ...
```
- **예상 시간**: 2-3시간
- **필요한 것**: 질문 30개 + 날짜 기반 선택 로직

### 7. "감정 테스트" 바이럴 기능 ⭐⭐⭐
```
"나의 숨겨진 감정 유형은?"
→ 5개 질문 → 결과: "당신은 '조용한 전사' 타입"
→ 예쁜 카드로 공유 → 친구도 해봄 → 다운로드
```
- **예상 시간**: 8-10시간
- **필요한 것**: 테스트 플로우 UI, 결과 유형 5-8개
