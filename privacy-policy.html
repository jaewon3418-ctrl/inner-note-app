1. 개인정보 수집 및 이용

DeepLog는 사용자의 감정 기록을 안전하게 보호합니다.

  • 수집 항목: 감정 일기 내용, 감정 상태, 작성 시간
  • 수집 목적: 감정 분석 및 맞춤형 위로 메시지 제공
  • 보관 기간: 사용자가 앱을 삭제할 때까지

2. 데이터 보안

  • 모든 데이터는 사용자 기기에만 저장됩니다
  • CTR+HMAC 암호화로 안전하게 보호됩니다
  • 외부 서버로 자동 전송되지 않습니다

3. OpenAI API 사용

⚠️ 중요: 감정 분석을 위해 OpenAI API를 사용합니다.
- 사용자가 명시적으로 동의한 경우에만 활성화
- 전송 데이터: 감정 일기 텍스트만 전송
- OpenAI 개인정보처리방침이 적용됩니다

4. 제3자 제공

사용자 동의 없이 개인정보를 제3자에게 제공하지 않습니다.

예외 사항:

  • 법령에 의한 요구가 있는 경우
  • 사용자가 명시적으로 동의한 경우 (OpenAI API 사용)

5. 사용자 권리

  • 언제든지 데이터 삭제 가능 (설정 → 모든 데이터 삭제)

  • OpenAI 동의 철회 가능
